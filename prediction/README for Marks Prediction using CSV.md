# CSV Data Analyzer with Linear Regression

##  Description
A Python-based tool that allows users to analyze any CSV dataset, visualize relationships between variables, and perform linear regression to predict outcomes.

This project demonstrates fundamental concepts of data analysis and machine learning using Pandas, Matplotlib, and Scikit-learn.

---

##  Features
- Load and analyze any CSV file
- Select independent (X) and dependent (Y) variables dynamically
- Display dataset overview (head, tail, shape, statistics)
- Visualize data using scatter plots
- Perform Linear Regression
- View slope, intercept, and model accuracy (R² score)
- Compare actual data with best-fit regression line
- Predict values based on user input
- Export predicted results to a new CSV file

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## How to Run

### 1. Install Dependencies
      pip install pandas matplotlib scikit-learn
### 2. Run the Program
      python Predictions_using_CSV.py
### 3. Steps to Use
Enter the file path of the CSV file
Choose:
Independent variable (e.g., Hours)
Dependent variable (e.g., Scores)
Select options from menu:
View data
Plot graph
Perform regression
Compare regression line
Predict values
